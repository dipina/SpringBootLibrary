\doxysubsection{Detailed Description}
Entity class representing a borrowing transaction between a user and a book. 

The {\ttfamily Borrowing} class maps to the "{}borrowings"{} table in the database. It captures the user, book, borrow date, and optional return date for each borrowing event.
