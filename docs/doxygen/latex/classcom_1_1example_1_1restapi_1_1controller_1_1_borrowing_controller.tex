Controller class providing RESTful endpoints for borrowing and returning books. 

This controller interacts with the {\ttfamily \doxylink{}{Borrowing\+Service}} and {\ttfamily \doxylink{}{User\+Service}} classes, managing book borrowing actions and retrieval of borrowed book data.
